# 2. Create GitHub Repo (IMPORTANT FIRST STEP)

Do this in the browser:

1. Go to https://github.com
2. Click **+ (top right)** → **New repository**
3. Enter repository name:
   - HowTo OR AstralAnomalies
4. Choose:
   - Public or Private
5. IMPORTANT SETTINGS:
   - DO NOT check “Add README”
   - DO NOT add .gitignore here (we manage it locally)
6. Click **Create repository**

Repeat once for each repo:
- HowTo
- AstralAnomalies

# 6. AstralAnomalies .gitignore (CRITICAL)

Create a file called:
.gitignore

Inside AstralAnomalies repo, include EXACTLY:
```
Library/
Temp/
Obj/
Build/
Builds/
Logs/
UserSettings/
```

---

## What this does
These folders are automatically generated by game engines and MUST NOT be uploaded.

If included:
- repo becomes huge
- builds break
- conflicts happen
- performance issues occur

# 10. .gitignore Rules (IMPORTANT)

Rules:
- Must be inside repo root
- Must be committed like normal file
- Only works on untracked files

If bad files were already added:

git rm -r --cached .
git add .
git commit -m "Fix gitignore"
